# Hotel Management System (Python Console Based)

## Project Overview

The Hotel Management System is a console based application written in Python.  
It helps manage hotel operations such as rooms, customers, bookings, billing, staff duties, and reports using file handling.

This project is developed for academic learning and practical understanding of core Python concepts.

---

## Features

### Admin
* Secure login
* Add and update rooms
* View all room details
* Create users and assign roles
* Assign worker duties
* Manage salaries
* View bookings and billing details
* Track attendance
* Generate reports
* Dashboard summary

### Receptionist
* Login authentication
* View available rooms
* Register customers
* Book rooms
* Checkout customers
* Generate bills

### Manager
* Login authentication
* View room information
* View booking records
* View billing reports
* View worker attendance and duties

### Worker
* Login authentication
* Check in for duty
* Check out after duty
* View assigned tasks
* View salary status

---

## Technologies Used

* Programming Language: Python  
* Interface: Console based  
* Data Storage: Text files using file handling  
* Libraries: Built in Python libraries  

---

## System Workflow

1. User logs in with username and password  
2. System validates credentials  
3. User role is identified  
4. Role based menu is displayed  
5. User performs allowed actions  
6. Data is saved in text files  
7. User logs out or exits
