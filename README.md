## Project Overview

The Hotel Management System is a comprehensive application designed to streamline and automate the operations of a hotel. It supports multiple interfaces including a Python Console, a Web Application, and a GUI, making it versatile for different usage scenarios.
It helps manage hotel operations such as rooms, customers, bookings, billing, staff duties, and reports using file handling.

This project is developed for academic learning and practical understanding of core Python concepts, web development, and GUI programming.

---

## Features

### Admin

- Secure login
- Add and update rooms
- View all room details
- Create users and assign roles
- Assign worker duties
- Manage salaries
- View bookings and billing details
- Track attendance
- Generate reports
- Dashboard summary

### Receptionist

- Login authentication
- View available rooms
- Register customers
- Book rooms
- Checkout customers
- Generate bills

### Manager

- Login authentication
- View room information
- View booking records
- View billing reports
- View worker attendance and duties

### Worker

- Login authentication
- Check in for duty
- Check out after duty
- View assigned tasks
- View salary status

## System Workflow

1. User logs in with username and password
2. System validates credentials
3. User role is identified
4. Role based menu is displayed
5. User performs allowed actions
6. Data is saved in text files/JSON
7. User logs out or exits

## How to Run the Project

1. Install Python on your system
2. Clone this repository
3. Open terminal or command prompt
4. Go to the project directory
5. Run the main file:
   ```bash
   python main.py
   ```
   Or for the web app:
   ```bash
   python app.py
   ```
